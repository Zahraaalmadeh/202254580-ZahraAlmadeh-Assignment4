# 📘 Technical Documentation

## 1. Overview

This project is a client-side personal web application built using **HTML, CSS, and JavaScript**.

It serves as a complete portfolio application and demonstrates all major course concepts, including:

- API integration  
- state management  
- advanced JavaScript logic  
- responsive design  
- user experience improvements  

---

## 2. Technologies Used

- HTML5 → Structure  
- CSS3 → Styling and responsiveness  
- JavaScript → Logic and interactivity  
- GitHub API → External data integration  
- localStorage → Persistent state  

---

## 3. Application Architecture

The application follows a simple front-end structure:
- index.html → UI structure  
- styles.css → layout and styling  
- script.js → logic and interaction  

--- 

## 4. Core Features
   ### 4.1 Theme Toggle
   Switch between light and dark mode
   Stored in localStorage
   Automatically restored on page reload
   ### 4.2 Project Filtering
   Projects are filtered using category attributes.
   Categories:
   AI
   Database
   Java
   GitHub
   ### 4.3 Project Sorting
   Sorting options:
   Name (A–Z)
   Name (Z–A)
   Newest
   Oldest
   Logic:
   Select visible projects
   Compare attributes
   Sort array
   Re-render UI
   ### 4.4 Contact Form Validation
   Validation includes:
   Required fields
   Valid email format
   Minimum message length
   Feedback messages guide the user.
   ### 4.5 GitHub API Integration
   Data is fetched using:
   https://api.github.com/users/Zahraaalmadeh/repos
   Behavior:
   Success → display repositories
   Failure → show fallback message
   ### 4.6 State Management
   Stored using localStorage:
   Theme preference
   Visitor name
   ### 4.7 Session Timer
   Tracks how long the user stays on the page.
   ### 4.8 Innovation Feature – Project Spotlight
   Randomly selects a project
   Highlights it
   Scrolls into view
   This adds interactivity and uniqueness.
