# AI Usage Report

## 1. Tools Used and Use Cases

### ChatGPT
I used ChatGPT as the main AI tool during Assignment 4. It helped me with:
- planning how to upgrade Assignment 3 into a more complete final application
- improving JavaScript logic such as filtering, sorting, validation, and spotlight behavior
- refining the UI/UX to make the website more professional and presentation-ready
- generating and improving documentation files
- helping structure the presentation outline and speaking script

---

## 2. Benefits of Using AI
Using AI helped me:
- generate ideas for polishing the final project
- improve the logic of JavaScript features
- make the portfolio feel more complete and professional
- write stronger documentation
- think more clearly about how to present the application in a 5–7 minute demo

AI was especially useful in identifying practical improvements that fit the assignment requirements without overcomplicating the project.

---

## 3. Challenges, Risks, and Limitations
AI assistance also introduced risks and limitations:

- some suggestions needed adjustment to match my exact codebase and project data
- AI-generated code may look correct but still require testing for edge cases
- some text suggestions can sound generic if used without rewriting
- overusing AI without understanding the output could weaken learning
- AI can propose more complexity than needed, so I had to keep the project focused and manageable

Because of these limitations, I reviewed and tested all code before keeping it in the final version.

---

## 4. Learning Outcomes
By using AI during Assignment 4, I learned:
- how to evolve an earlier portfolio into a more polished final application
- how to add small innovation features without making the application unstable
- how to better organize user guidance and presentation-ready features
- how to improve technical writing and documentation structure
- how to evaluate AI suggestions instead of accepting them automatically

AI helped me improve both the project itself and my workflow.

---

## 5. Responsible Use and Modifications
I used AI responsibly by:
- reviewing each suggestion before applying it
- modifying the generated code so it matched my own project and real portfolio content
- testing functionality manually after important changes
- adapting documentation so it reflected my own understanding
- using AI for support, refinement, and learning rather than copying blindly

I made sure the final submission reflects my own work, choices, and understanding.

---

## 6. Conclusion
AI tools played a helpful supporting role in Assignment 4. They improved development speed, documentation quality, and idea generation, but I remained responsible for reviewing, understanding, testing, and adapting the final result. This helped me complete the assignment in a way that supports both learning and academic integrity.
