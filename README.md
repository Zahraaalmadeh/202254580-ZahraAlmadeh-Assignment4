# Zahra Almadeh Portfolio – Assignment 4

## 📌 Project Description

This project is a polished personal web application developed for **Assignment 4 – Personal Web Application**.

It represents the final version of my portfolio website, combining all concepts learned in the course, including responsive design, JavaScript interactivity, API integration, state management, and user-centered design.

The goal of this project is to showcase my skills, academic projects, and technical abilities in a professional and presentation-ready format.

---

## 🚀 Features

### 🎯 Core Portfolio Features
- Responsive layout for desktop and mobile
- Sticky navigation bar
- Professional hero section
- Resume download button
- Contact form with validation

### 🧠 Skills & Projects
- Skills section highlighting technical abilities
- Real academic and technical projects
- Project filtering by category
- Project sorting by name and date

### ⚡ Advanced Functionality
- GitHub API integration (live repositories)
- Dark/light mode with persistent state
- Visitor name personalization (localStorage)
- Session timer showing time on site
- Project Spotlight feature (random highlight)

### 👤 User Experience
- Clear instructions for users
- Friendly validation messages
- Error handling and fallback messages
- Accessible and responsive design
