# Zahra Almadeh Portfolio – Assignment 4

## Project Description
This project is a polished personal web application developed for **Assignment 4 – Personal Web Application**. It builds on the previous portfolio assignments and brings all required concepts together into a more complete, professional, and presentation-ready application.

The website showcases my background, skills, academic projects, and GitHub repositories in a clean and responsive interface built with **HTML, CSS, and JavaScript**.

---

## Main Features

### Personal Portfolio Features
- Responsive personal portfolio layout
- Sticky navigation bar
- Professional hero section
- Downloadable resume button
- Contact section with user-friendly validation

### Skills and Project Showcase
- Skills section highlighting technical strengths
- Real academic and technical projects
- Project filtering by category
- Project sorting by name and date
- Friendly empty-state message when no projects match

### Dynamic and Advanced Features
- GitHub API integration for live repositories
- Dark/light mode with saved preference
- Visitor name greeting stored in `localStorage`
- Session timer showing time on site
- Project Spotlight feature that randomly highlights a visible project

### User Experience
- Helpful UI guidance
- Clear fallback messages
- Accessible navigation and form feedback
- Responsive design across screen sizes
